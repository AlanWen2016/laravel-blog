## About laravel-blog

This is back-end system base on Laravel framework 5.5.

## Feature

- Api for vue-blog
- QQ login

### Start


```
  git clone https://github.com/AlanWen2016/laravel-blog.git
```

install vendor
```
  composer install
```
```
php artisan key:generate
```

migrate Database

```
php artisan migrate
```
